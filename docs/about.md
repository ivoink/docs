# Inkwell Ops

> 打造一个能让真心喜欢动手，自己学习与部署服务的小伙伴在我的基础上少走许多的弯路，快速入门

**Inkwell Ops** ，一个从 Linux 基础命令到网络设备，打造完整的自托管部署指南


## 内容方向

### Linux

从文件系统、用户权限到进程管理和网络配置，系统讲解 Linux 操作系统的核心知识。理论与实验相结合，注重常用命令与实际场景的对应，帮助读者快速建立对操作系统的整体认知，并为后续服务部署打下基础。同时涵盖 Shell 脚本入门、环境变量与管道重定向等实用内容。

### 自托管服务部署

围绕 Docker 容器化方案展开，覆盖私有网盘、密码管理器、AI（OpenClaw）等常见自建服务的安装与配置，提供可复现的部署流程和排错思路。

### 公网访问方案

针对家庭或内网服务的外部访问需求，介绍内网穿透的方法与工具选型，并给出免费方案的实操步骤，让本地服务能够安全地对外提供访问。

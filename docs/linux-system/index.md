# Linux

本部分是我在日常学习和探索 Linux 系统时整理的知识库文档，包含实战操作与理论讲解。

## 概览

通过实际操作与理论结合的方式，系统掌握 Linux 操作系统的核心知识点，快速上手常用命令与服务部署。

## 主要内容

| 章节 | 标题 | 内容简介 |
| :---: | :---: | :---: |
| 01 | Linux 安装与远程登录 | Ubuntu Server 安装配置、VMware 虚拟机搭建、SSH 远程连接与常用客户端使用 |
| 02 | 软件与软件源 | apt/dpkg 包管理器使用、软件源配置、国内镜像源切换与 DEB822 格式解析 |
| 03 | 用户类型和用户权限 | sudo 提权机制、sudoers 配置、用户与用户组管理、chmod/chown 权限操作 |
| 04 | 文件编辑与操作 | vim 编辑器使用、文件/目录操作命令、软硬链接创建与管理 |
| 05 | 系统与服务 | systemd 服务管理、常用系统服务配置、日志查看与系统状态监控 |
| 06 | Linux 系统配置与安全 | 网络配置、时间同步、防火墙设置、系统安全加固与 SSH 密钥认证 |
| 07 | 网络配置与管理 | IP 地址配置、网络诊断工具、路由表查看、DNS 解析与网络问题排查 |
| 08 | 进程管理与日志排查 | 进程查看与终止、日志管理、journalctl 使用、异常进程分析与排查 |
| 09 | 磁盘与存储管理 | 磁盘分区、文件系统挂载、存储空间查看、LVM 逻辑卷管理 |
| 10 | 定时任务 | cron 定时任务配置、系统级定时任务、定时脚本编写与日志管理 |

## 学习目标

- 完成 Ubuntu Server 安装与网络配置
- 熟练使用 SSH 客户端远程管理服务器
- 理解 Linux 用户权限模型，掌握用户管理与文件权限操作
- 为后续服务部署、环境搭建打下基础

::: tip 提示
本文档基于 Ubuntu Server 24.04.3 LTS 进行演示，其他 Debian 系发行版可参考。
:::
